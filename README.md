# 🛒 Customer Purchase Prediction & Recommendation System

## 📌 Overview

This project is an end-to-end machine learning solution designed to predict customer purchase likelihood and recommend relevant products based on user behavior and transaction data.

---

## 🚀 Features

* Predicts purchase probability using ML models like XGBoost and Random Forest
* Performs feature engineering on session activity, cart behavior, and purchase history
* Implements a hybrid recommendation system using collaborative filtering and K-Means clustering
* Improves customer targeting and conversion rates

---

## 🛠️ Tech Stack

* Python (Pandas, NumPy, Scikit-learn, XGBoost)
* Machine Learning
* Flask (API deployment)
* Clustering (K-Means)

---

## 📊 Model Performance

* Accuracy: ~87%
* ROC-AUC: ~0.90
* Evaluated using Precision, Recall, and F1-score

---

## ▶️ How to Run

```bash
pip install -r requirements.txt
python src/train.py
python app/app.py
```

## 💡 Use Case

Helps businesses identify high-value customers, personalize recommendations, and optimize marketing strategies.
